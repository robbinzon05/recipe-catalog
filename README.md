# Recipe Catalog

Веб-приложение на Django для каталога рецептов.

## Стек

- Python 3
- Django
- SQLite
- Bootstrap 5

## Быстрый запуск

```powershell
python -m venv .venv
.\.venv\Scripts\pip.exe install -r requirements.txt
.\.venv\Scripts\python.exe manage.py runserver
```

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:8000/
```
