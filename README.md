# Recipe Catalog

Веб-приложение на Django для каталога рецептов. Пользователи могут просматривать рецепты, искать блюда, фильтровать каталог по категориям и оставлять комментарии после входа в систему. Администратор управляет рецептами, категориями и комментариями через Django Admin.

## Стек

- Python 3.13
- Django
- SQLite
- Bootstrap 5

## Возможности

- главная страница с подборкой рецептов;
- список рецептов с пагинацией;
- фильтрация по категориям;
- поиск по названию, описанию и ингредиентам;
- детальная страница рецепта;
- регистрация, вход и выход;
- комментарии авторизованных пользователей;
- редактирование и удаление своих комментариев;
- административная панель для staff/superuser.

## Быстрый запуск

```powershell
python -m venv .venv
.\.venv\Scripts\pip.exe install -r requirements.txt
.\.venv\Scripts\python.exe manage.py migrate
.\.venv\Scripts\python.exe manage.py loaddata initial_data
.\.venv\Scripts\python.exe manage.py runserver
```

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:8000/
```

## Администратор

Создать администратора можно командой:

```powershell
.\.venv\Scripts\python.exe manage.py createsuperuser
```

Административная панель:

```text
http://127.0.0.1:8000/admin/
```

Обычные пользователи не имеют доступа к административным действиям.

## Тесты

```powershell
.\.venv\Scripts\python.exe manage.py test
```

## Структура базы данных

Основные сущности:

- `Category` - категория рецепта;
- `Recipe` - рецепт с описанием, ингредиентами, шагами и изображением;
- `Comment` - комментарий пользователя к рецепту;
- `User` - стандартная модель пользователя Django.
